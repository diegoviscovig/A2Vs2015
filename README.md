# A2Vs2015 - Angular 2 Visual Studio 2015

**StarterKit**

**Include**
* Angular 2 Router
* Bootstrap SASS + Font Awesome
* Gulp 3 and rebuild on changes
* Browser Sync

**Styles SASS**
* Include Bootsrap
* File /css/app.scss

**Start**
* cd ./src/A2Vs2015
* npm install
* gulp serve
* Open A2Vs2015 Solution in Visual Studio 2015 and edit files
